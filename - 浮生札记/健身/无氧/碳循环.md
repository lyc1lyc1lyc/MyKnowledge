
多维和鱼油随“正餐（有油）”走，肌酸随“碳水（米饭）”走

**胸 -> 腿 -> 背 -> 休息**。

1. **（胸）：** 中碳（500g 米饭总配额）。
    碳水225g，蛋白质150g，脂肪60g
    
2. **（腿/背）：** 高碳（800g 米饭总配额，严格控油，不要蛋黄）。
    碳水306g，蛋白质150g，脂肪48g
    
3. **（休息）：** 低碳（250g 米饭配额，其余靠豆浆）。
    碳水130g，蛋白质150g，脂肪75g
    

| **训练顺序**      | **碳水档位** | **碳水 (C)** | **蛋白质 (P)** | **脂肪 (F)** | **核心饮食逻辑 (以熟米饭为例)**    |
| ------------- | -------- | ---------- | ----------- | ---------- | ---------------------- |
| **Day 1: 胸**  | **中碳日**  | **225g**   | 150g        | 60g        | **500g 熟米饭** (午晚各250g) |
| **Day 2: 腿**  | **高碳日**  | **320g**   | 150g        | 45g        | **800g 熟米饭** (午晚各400g) |
| **Day 3: 背**  | **高碳日**  | **320g**   | 150g        | 45g        | **800g 熟米饭** (午晚各400g) |
| **Day 4: 休息** | **低碳日**  | **130g**   | 150g        | 75g        | **250g 熟米饭** (仅午餐吃)    |


| **档位**  | **对应训练**    | **碳水 (g/kg)** | **蛋白 (g/kg)** | **脂肪 (g/kg)** |
| ------- | ----------- | ------------- | ------------- | ------------- |
| **高碳日** | **腿 / 背**   | **4.5 - 5.0** | **2.2**       | **0.7**       |
| **中碳日** | **胸 (卧推日)** | **3.3**       | **2.2**       | **0.9**       |
| **低碳日** | **休息日**     | **1.8 - 2.0** | **2.2**       | **1.1**       |

| **状态**  | **碳水 (C)**                           | **蛋白质 (P)**                          | **脂肪 (F)**                          | **建议训练** |
| ------- | ------------------------------------ | ------------------------------------ | ----------------------------------- | -------- |
| **高碳日** | **$4.5\text{g/kg}$ ($306\text{g}$)** | **$2.2\text{g/kg}$ ($150\text{g}$)** | **$0.7\text{g/kg}$ ($48\text{g}$)** | 练腿 / 练背  |
| **中碳日** | **$3.3\text{g/kg}$ ($225\text{g}$)** | **$2.2\text{g/kg}$ ($150\text{g}$)** | **$0.9\text{g/kg}$ ($60\text{g}$)** | 练胸       |
| **低碳日** | **$1.9\text{g/kg}$ ($130\text{g}$)** | **$2.2\text{g/kg}$ ($150\text{g}$)** | **$1.1\text{g/kg}$ ($75\text{g}$)** | 休息日      |
