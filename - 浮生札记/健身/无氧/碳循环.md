
**胸 -> 腿 -> 背 -> 休息**。

1. **（胸）：** 中碳（500g 米饭总配额）。
    碳水225g，蛋白质150g，脂肪60g
    
2. **（腿/背）：** 高碳（800g 米饭总配额，严格控油，不要蛋黄）。
    碳水306g，蛋白质150g，脂肪48g
    
3. **（休息）：** 低碳（250g 米饭配额，其余靠豆浆）。
    碳水130g，蛋白质150g，脂肪75g
    
#### **Day 1：中碳日 (胸部训练日 )**

- **08:30-10:00:** 100g 干豆豆浆（练前/中，提供持续氮平衡）
    
- **10:15:** **3 个熟鸡蛋白 (弃黄)**（练后快速补丁）+ 15min 坡度快走
    
- **12:30 (午餐):** 125g 鸡胸肉 + **250g 熟米饭** + 炒菜 (控油)
    
- **18:30 (晚餐):** 125g 鸡胸肉 + **250g 熟米饭** + 炒菜 (控油)
    
- **22:30 (睡前):** 50g 干豆豆浆糊 + 1 勺蛋白粉 (暖气补丁，解决冷感)
    
- **全天指标：** 碳水 225g / 蛋白 150g / 脂肪 60g
    

#### **Day 2：高碳日 (腿部训练日)**

- **08:30-10:00:** 100g 干豆豆浆（练前/中）
    
- **10:15:** **4 个熟鸡蛋白 (弃黄)**（大肌群损伤修复）
    
- **12:30 (午餐):** 125g 鸡胸肉 + **400g 熟米饭** + 炒菜 (极简油/水煎)
    
- **18:30 (晚餐):** 125g 鸡胸肉 + **400g 熟米饭** + 炒菜 (极简油/水煎)
    
- **22:30 (睡前):** **1.5 勺蛋白粉 + 30g 燕麦** (取消干豆以腾出脂肪配额给碳水，同时燕麦维持热效应)
    
- **全天指标：** 碳水 320g / 蛋白 150g / 脂肪 45g (严控油)
    

#### **Day 3：高碳日 (背部训练日 - 引体/划船)**

- **08:30-10:00:** 100g 干豆豆浆（练前/中）
    
- **10:15:** **4 个熟鸡蛋白 (弃黄)** + 15min 坡度快走
    
- **12:30 (午餐):** 125g 鸡胸肉 + **400g 熟米饭** + 炒菜 (极简油)
    
- **18:30 (晚餐):** 125g 鸡胸肉 + **400g 熟米饭** + 炒菜 (极简油)
    
- **22:30 (睡前):** **1.5 勺蛋白粉 + 30g 燕麦** (保持高碳合成环境，对抗深夜分解)
    
- **全天指标：** 碳水 320g / 蛋白 150g / 脂肪 45g
    

#### **Day 4：低碳日 (休息日 - 主动恢复)**

- **08:30:** 100g 干豆豆浆 (开启全天代谢)
    
- **10:30:** **15-20min 坡度快走** (主动恢复，改善循环)
    
- **12:30 (午餐):** 125g 鸡胸肉 + **250g 熟米饭** + **3 个全蛋 (含蛋黄)** + 炒菜 (可正常放油)
    
- **18:30 (晚餐):** 125g 鸡胸肉 + **菜** (不吃主食，拉高燃脂效率)
    
- **22:30 (睡前):** 50g 干豆豆浆糊 + 1 勺蛋白粉 (高脂慢蛋白供暖，对抗低碳冷感)
    
- **全天指标：** 碳水 130g (大脑底线) / 蛋白 150g / 脂肪 75g (激素修复)
