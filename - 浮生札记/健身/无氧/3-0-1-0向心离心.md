
“凡是举起东西的，用力快（向心，1秒）；凡是放下东西的，控制慢（离心，3秒）。”
向心收缩肌肉，离心拉伸肌肉

### “3-0-1-0”
这个节奏是职业健美运动员和力量选手的“秘密武器”，目的是**彻底压榨肌肉张力**。

- **3 秒离心（下放）**：想象你是一个液压机，杠铃不是掉下来的，而是被你“压”下来的。在这个过程中，你必须全程紧绷胸大肌，感受纤维被一点点拉长。**这是为了产生最多的肌肉肥大信号。**
    **补充说明：**若感觉某次训练状态不佳，爆发不起来，可将离心缩短至 **1.5-2 秒**，并在触胸瞬间利用肌肉弹性能量（SSC）快速推起
- **0 秒底部停顿**：底部不要放松，不要让杠铃借着胸口的弹性跳起来。
    
- **1 秒向心（爆发推起）**：这是神经系统的爆发时刻。不需要慢，要果断、迅速，以最快速度把重量推起，带动肌纤维全力收缩。
    
- **0 秒顶部停顿**：不要在最上面锁死关节休息。如果锁死，重力就压在骨骼上了，肌肉就没张力了。保持关节微屈，立刻开始下一个 3 秒离心。
    资料库补充：如果在顶部完全放松（尤其是推举类），重量会压在骨骼和结缔组织上，增加关节受伤风险。




1. **向心收缩（Concentric）**：
    - **定义**：肌肉缩短，产生力量克服阻力（如卧推将杠铃**推起**的过程）。
    - **目标**：通常要求尽可能**爆发性、快速**地发力，以动员更多的高阈值快肌纤维，从而提升力量和爆发力。
2. **离心收缩（Eccentric）**：
    - **定义**：肌肉在受压状态下被拉长（如卧推**慢放**杠铃的过程）。
    - **作用**：离心过程不仅能产生微损伤促进增肌，更重要的是**储存弹性势能**
